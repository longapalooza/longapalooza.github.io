psypy

psypy is a python library for calculation psychrometric states of moist air.

Source: https://github.com/longapalooza/psypy

matheval

matheval is a python class to evaluate basic ASCII math syntax. This is a port from a PHP project to Python. The project can be found here:
https://github.com/Itangalo/evalmath.class.php.
That project is originally based on code written by Miles Kaufmann. Miles Kaufmann is in no way affiliated with this project.

Source: https://github.com/longapalooza/matheval

MUA

MUA is a web application that performs common uncertainty analysis tasks using modular uncertainty techniques.

Source: https://github.com/longapalooza/MUA

Application: https://longapalooza.github.io/MUA
